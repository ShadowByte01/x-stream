## ✨ About The Project

**X-Stream** is a stunningly designed, high-performance streaming application that allows users to discover, stream, and enjoy their favorite movies and TV series effortlessly. Engineered with modern web technologies, it features blazing-fast load times, seamless authentication, and an intuitive user interface.

## 🚀 How It Works

X-Stream brings a flawless streaming experience through a modern tech stack:
* **Frontend:** Built with React and Vite for a lightning-fast, reactive user experience.
* **Authentication:** Integrated with Supabase, offering secure email/password signups and logins, fortified by Cloudflare Turnstile CAPTCHA.
* **Database & Storage:** Powered by Supabase for real-time data syncing and media asset storage.
* **Hosting:** Deployed seamlessly on Vercel for instantaneous global delivery.

## 🛠️ Installation & Setup

Want to run X-Stream locally? Follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/x-stream.git
   cd x-stream
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file in the root directory and add your Supabase and CAPTCHA keys:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   VITE_CAPTCHA_SITE_KEY=your_captcha_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
